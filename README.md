# Daily GenAI & AI Agents News

An automated daily news digest covering **Generative AI** and **AI Agents** — generated by a scheduled Cowork task and compiled into a formatted Word document each day.

## 📁 Folder Structure

```
daily-genai-news/
├── outputs/                     ← Daily .docx reports saved here
│   └── GenAI-News-YYYY-MM-DD.docx
├── create_news_doc.js           ← Script that generates the Word document
├── package.json                 ← Node.js dependencies
├── .gitignore                   ← Ignores node_modules and temp files
└── README.md                    ← You are here
```

## 📰 What's Covered Each Day

| Section | Topics |
|---|---|
| 🏆 Top Story | The single most important AI development of the day |
| 🤖 Model Releases | New models, fine-tunes, capability updates |
| 🔬 Research Highlights | Notable papers and technical breakthroughs |
| 🕵️ AI Agents | Agent frameworks, multi-agent systems, autonomous AI |
| ⚙️ Training & Inference | Efficiency, hardware, deployment advances |
| 💼 Industry & Business | Funding, acquisitions, partnerships, policy |
| 👀 Worth Watching | Emerging trends to keep an eye on |

## 🚀 How It Works

1. A scheduled Cowork task runs daily
2. It searches the web for the latest AI news across multiple topics
3. Results are compiled and formatted into a `.docx` report
4. The report is saved to the `outputs/` folder

## 🛠️ Running Manually

```bash
npm install
node create_news_doc.js
```

The report will be saved to `outputs/GenAI-News-YYYY-MM-DD.docx`.

## 📦 Dependencies

- [docx](https://www.npmjs.com/package/docx) — Word document generation

## 📅 Report Archive

Reports are committed daily and browseable by date in the `outputs/` folder.
